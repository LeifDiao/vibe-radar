Key Features

Three layers of evaluation

ClaudeRadar grades how you talk to AI, how you set up the platform, and what you actually ship.

9 dimensions × 3 categories

Communication — how clearly you direct AI. Engineering — your platform leverage (Skill / MCP / Subagent / CLAUDE.md / Plan mode). Outcome — what actually gets shipped (efficiency, verification, clean closure).

AI diagnosis layer

Free-form collaboration profile + core diagnosis (strength + bottleneck) + cross-dimension reading. This is the part users find most valuable.

Professional, readable dashboard

Single-file HTML dashboard built for readability — scores, diagnosis, and pastable suggestions in one place, easy to scan and easy to share.

The Core Value

AI writes the diagnosis

Scores tell you what. Diagnosis tells you why and what to do next. Independent of scoring, evidence-grounded, no personality archetypes.

CORE DIAGNOSIS
"Strength: Your Efficiency + Completion combo is top-tier — work actually ships, sessions close cleanly. Bottleneck: Across 8 sessions you almost never asked AI to explain its plan before executing."
PATTERN READING
"High Lock-On + low Proof Check = you trust AI's execution but not its judgment. Fine for tight, well-scoped work; risky when complexity climbs."
PASTABLE PROMPT REWRITE
"Before you write any code, walk me through what could go wrong with this change and how you'd test it."
Every improvement suggestion ships with a concrete prompt you can copy into your next session, plus expected score impact.
Fairness

A 3-message bugfix is not a 50-session build

Every project is auto-classified into one of 4 profiles, each with its own dimension weights and N/A rules. Small projects no longer get unfairly penalized.

AOne-shot · ≤ 2 sessions, ≤ 15 msgs
BFeature build · 3-20 sessions, balanced edit ratio
CLong-running · 20+ sessions or > 7 days
DLearning · Q&A-heavy, few edits
Density beats volume. A 5-message project with high signal density and 3 file edits gets high confidence and full marks for Efficiency. Confidence scaling rewards signal-rich sessions, not just session count.
